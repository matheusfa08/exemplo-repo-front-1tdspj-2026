```
1 - Expandir a atividade apresentada criando novas páginas dentro da pasta original

3 - Determine um tema para todas as páginas. Incluindo as páginas existentes. Com tema quer dizer que vamos falar sobre, por exemplo: esportes, tigrinho, etc. Então teremos a página de futebo, de basquete, etc.

8 - Para a imagem, utilize a tag <img scr="/imagem.png" alt ="descrição" width="largura" heigth="altura">
9 - Para cada página, faça um commit.
10 - Faça um push e um PR, atualize a pages do git e verifique o resultado.
